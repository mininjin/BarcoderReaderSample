# バーコードーリーダーテストアプリ

QuaggaJsで作ったバーコードリーダーアプリです。カメラでJANコードを読み取ってその番号を表示できます。

## セットアップとサーバー起動
```
npm install
npm run serve
```

## 対応確認機種・OS
iOS以外は古い機種が多いですがご容赦ください。。

Android: Galaxy S20, Xperia 5, AQUOS SH-02J
iPhone：iPhoneSE, iPhone8, iPhoneSE2, iPhone12 (それぞれiOS14, 15)
